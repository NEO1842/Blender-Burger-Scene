# Security Policy / セキュリティポリシー

---English---

## Supported Versions

| Version | Supported |
|---|---|
| Latest | ✅ |
| Older versions | ❌ |

## Reporting a Vulnerability

This project is a Blender scene file and does not contain executable code or handle sensitive data.  
However, if you discover any security concerns related to this repository (e.g. malicious files, data exposure), please report them by:

- Opening a **private** issue, or
- Contacting the repository owner directly via GitHub

Please do **not** open a public issue for security vulnerabilities.

## 脆弱性の報告

このプロジェクトはBlenderのシーンファイルであり、実行可能なコードや機密データを含みません。  
ただし、リポジトリに関するセキュリティ上の懸念（悪意のあるファイル・データの漏洩など）を発見した場合は、以下の方法でご報告ください：

- **非公開** のIssueを開く、または
- GitHubからリポジトリオーナーに直接連絡する

セキュリティの脆弱性について、公開Issueを開くことは**しないでください**。
