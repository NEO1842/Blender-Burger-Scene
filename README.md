# Blender-Burger-Scene

A stylized 3D burger scene created in Blender.

# Overview

This project is a Blender scene featuring a fully modeled cheeseburger — complete with a sesame seed bun, lettuce, tomato, cheese, and beef patty, served on a clean white plate.

It uses warm and soft lighting to give the food an appetizing, delicious look.

The scene can be freely modified and used for learning food modeling, material shading, and lighting workflows in Blender.

# Software Requirement

Blender Version 3.6 LTS

# Rendering Settings

Engine: Cycles

Sampling: 1024

Denoising: None

# Blender バーガーシーン

Blenderで作成されたスタイライズされた3Dバーガーシーンです。

# 概要

このプロジェクトは、チーズバーガーを完全にモデリングしたBlenderシーンです。  
ごまバンズ、レタス、トマト、チーズ、パティを備えた本格的な構成で、白いお皿に盛り付けています。

ウォームでソフトなライティングを使い、美味しそうで魅力的な見た目に仕上げています。

Blenderでの食べ物モデリング・マテリアル・ライティングの学習ベースとして自由に改変して使用できます。

# ソフトウェア要件

Blender バージョン 3.6 LTS
