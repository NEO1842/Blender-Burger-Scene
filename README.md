## Overview

A stylized 3D burger scene created in Blender.  
Features a fully modeled cheeseburger with sesame seed bun, lettuce, tomato, cheese, and beef patty — served on a clean white plate.

The scene uses a warm, soft lighting setup to give the food a delicious, appetizing look.  
Great for learning food modeling, material shading, and lighting in Blender.

---

## Software Requirement

- Blender Version **3.4** (recommended) or 3.6 LTS

---

| Setting | Value |

| Engine | Cycles |
| Sampling | 1024 |

## 概要

Blenderで制作した、スタイライズされた3Dバーガーシーンです。  
ごまバンズ、レタス、トマト、チーズ、パティを備えた本格的なチーズバーガーを白いお皿に盛り付けています。

ウォームでソフトなライティングを使い、美味しそうで魅力的な見た目に仕上げています。  
Blenderでの食べ物モデリング・マテリアル・ライティングの学習に最適です。
